应该是需要逆这个加密算法的进行求解，密码手的老本行了，写个求解脚本:

```python
import base64

def Redecode(cipher):
    message = ''
    cipher = base64.b64decode(cipher).decode()
    for i in cipher:
        x = ord(i) - 16
        x = x ^ 32
        message += chr(x)

    return message

cipher = 'XlNkVmtUI1MgXWBZXCFeKY+AaXNt'

flag = Redecode(cipher)
print(flag)
```

在python2环境下运行脚本，就能得到flag了：

